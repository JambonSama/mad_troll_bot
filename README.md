# Mad Troll Bot

Originally written in js because I didn't know better at the time. Might rewrite in python only in the future.

1. Create [your bot](https://discordapp.com/developers/applications/ "Create your bot")
1. Get the correct bot token, from the same URL, on your bot page, under the "bot" tab.
	* Text permissions
		* read message history
	* Voice permissions
		* connect
		* video
		* speak
	* General permissions
		* view channels
1. Insert the token in the index.js file
1. Invite the bot in the target server
1. Launch the code (necessitate the [discord.js node](https://discord.js.org/#/ "discord.js node") for now, while in js)
1. Write the prompt in the general channel and have fun
