# Mad Troll Bot

Written in js because I didn't know better at the time. Might rewrite in python only.

1. Create your bot [ https://discordapp.com/developers/applications/ ]
1. Get the correct bot token [ https://discordapp.com/developers/applications/ name of your bot /bot ]
	* Text permissions
		* read message history
	* Voice permissions
		* connect
		* video
		* speak
	* General permissions
		* view channels
1. Insert the token in the index.js file
1. Invite the bot in the target server
1. Launch the code (necessitate the discord.js node [ https://discord.js.org/#/ ] for now, while in js)
1. Write the prompt in the general channel and have fun
